┌──────────────────────────────────────────────────────────────────────────────┐
│ 🔄 WORKFLOW COMPARISON                                                       │
└──────────────────────────────────────────────────────────────────────────────┘

NORMAL EXAM (unchanged):
  1. Go to qsend.php
  2. Select "Normal Exam (All Centers)"
  3. Choose session & day
  4. Enter subject
  5. Click "Send Questions to All Centers"
  → Sends to ALL centres from DEASemester folder

MAKEUP EXAM (new):
  1. Upload PDFs to custom folder (manage_questions.php)
  2. Go to qsend.php
  3. Select "Makeup Exam (Specific Centers)"
  4. Choose question folder
  5. Choose session & day
  6. Select specific centres
  7. Enter subject
  8. Click "Send Questions to Selected Centers"
  → Sends ONLY to selected centres from chosen folder

Example 1: Upload 500 Biology PDFs
  → Go to manage_questions.php
  → Select "Makeup Exam"
  → Folder name: "Biology_Makeup_Jan2024"
  → Choose "Upload ZIP File"
  → Upload your biology.zip
  → Done in ~2 minutes!

Example 2: Send to Lagos & Abuja Only
  → Go to qsend.php
  → Select "Makeup Exam"
  → Folder: "Biology_Makeup_Jan2024"
  → Session & Day
  → Check Lagos & Abuja centres only
  → Send!

Example 3: Delete Wrong Files
  → Go to semitems.php?folder=DEASemester/Biology_Makeup_Jan2024
  → Click Delete on wrong files
  → Confirm deletion
  → Done!
